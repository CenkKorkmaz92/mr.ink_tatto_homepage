# Mr. Ink Tattoo - Official Homepage

Professional website for Mr. Ink Tattoo, a premier tattoo and piercing studio specializing in custom artwork and body art.

## About

Mr. Ink Tattoo offers professional tattoo and piercing services with a focus on quality, hygiene, and artistic excellence. Our studio features experienced artists who create unique, custom designs tailored to each client's vision.

## Features

- **Portfolio Gallery** - Showcase of tattoo work and designs
- **Artist Profiles** - Meet our talented team
- **Pricing Information** - Transparent pricing for services
- **Contact Form** - Easy booking and inquiries
- **Responsive Design** - Optimized for all devices
- **Multi-language Support** - Available in English and German

## Technologies

- **React 18** - Modern UI framework
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first styling
- **shadcn/ui** - High-quality component library
- **React Router** - Navigation and routing

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/     # Reusable UI components
├── pages/         # Page components
├── assets/        # Images and media
├── hooks/         # Custom React hooks
├── lib/           # Utility functions
└── main.tsx       # Application entry point
```
